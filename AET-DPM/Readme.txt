Specifications

 *   7107 based Digital Panel Meter 200mV / 2V
 *   Precision rectifier for AC to DC Conversion
 *   Decimal Point Selection
 *   Provision for variable reference
 *   Dual Supply Operated (±5V)
 *   Built-in reference
 *   Can be interfaced with various projects for measurement of : AC / DC Volt, Current, Resistance, pH, Conductivity, Dwell Angle, Engine RPM, Frequency Counter, Digital Weign Scale


Operation

AET - DPM Kit


1) Connect the ± 5V as described in setup_diagram.
2) Connect voltage to be measure (0-200mV).
3) See the result on seven segment LED display.
